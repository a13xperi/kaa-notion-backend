## Request Template

When creating a new request, copy this template:

```markdown
### REQ-[YYYYMMDD-HHMM]: [Title]
- **From:** Agent [N]
- **To:** Agent [M]
- **Need:** [Specific request - what do you need?]
- **Files Affected:** [paths that would need to change]
- **Priority:** BLOCKING | HIGH | NORMAL | LOW
- **Status:** PENDING | IN PROGRESS | COMPLETE
- **Notes:** [Any additional context]
```

### Priority Definitions

- **BLOCKING** - Cannot proceed without this, highest urgency
- **HIGH** - Significantly impacts progress, should be addressed soon
- **NORMAL** - Standard request, address when convenient
- **LOW** - Nice to have, no rush
