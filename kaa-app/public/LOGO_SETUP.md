# SAGE Logo Setup

## Logo File Location

Place the SAGE Garden Wizard logo image file at:

```
kaa-app/public/sage-logo.png
```

## Logo Specifications

- **File Name**: `sage-logo.png`
- **Recommended Format**: PNG with transparency
- **Recommended Size**: 512x512px (or higher resolution for crisp display)
- **Aspect Ratio**: 1:1 (square)

## Logo Usage

The logo is integrated into the following components:

1. **Landing Page** - Large logo with text
2. **Client Portal Landing** - Large logo with text
3. **Client Workspace Header** - Small logo in navigation bar
4. **Client Hub Hero Section** - Medium logo in welcome banner
